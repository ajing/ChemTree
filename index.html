ChemTree

Hierachical tree visualization of a molecule library

Download this project as a .zip file Download this project as a tar.gz file

Introduction

ChemTree is an Open Source interactive application for visualizing chemical similarity/diversity in a molecule library. ChemTree is built almost entirely of AngularJS and D3.

Features

  • Server side platform neutral implementation. Can be ported to PHP, JSP easily.
