ChemTree

D3 based phylogenetic-like tree visualization for a molecule library.

Download .zip Download .tar.gz View on GitHub

The following examples will work best in Chrome.

Key Features.

  • Gravity,Links
  • Semantic Zoom
  • Multiple filtering

Interesting Examples.

  • Map Activity to Rainbow colors or Black/White
  • 500 allosteric + 500 competitive
  • 500 allosteric + 500 competitive with link length
  • Compare two different hierarchies
  • Towards 100K Molecules
